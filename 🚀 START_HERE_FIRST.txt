🆘 TROUBLESHOOTING:

Problem: "Docker is not running"
Solution: Start Docker Desktop app, wait for it to say "Running"

Problem: "Python not found"
Solution: Download from python.org, check "Add to PATH" during install

Problem: "npm not found"
Solution: Reinstall Node.js from nodejs.org

Problem: Setup takes too long
Answer: Normal! First time downloads images (5-10 min)

Problem: Nothing happens when I double-click
Windows: Right-click → "Run as administrator"
Mac: Right-click → Open (allow unsigned apps)
